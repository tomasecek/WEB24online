
Zadani bootstrap1 (framework pro HTML, CSS a JavaScript)

* zalozte ve slozce web24 slozku bootstrap1 a nakopirujte soubory do slozky bootstrap1 soubory
  Provedte commit s nazvem „initfolder“.

* vytvorte si zastupce v prohlizeci pro snadne lokalni prohlizeni (prace pri hodine)

* prohlednete si vzor, postupujte dle vykladu (validator atp.)

* napojte soubor index.html na bootstrap.css

     <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css"
           rel="stylesheet"
           integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN"
           crossorigin="anonymous">

* změnte title stranky  <title>První stránka s bootstrap</title>

* nepouzivejte obrazky (klasifikace o stupen).

* v souboru index.html se vyvarujte primeho pouziti parametru style (klasifikace o stupen).

* pouzijte class z bootstrap (nezavádějte vlastní style, klasifikace o stupen)

* doplňta napojení index.html na web 24.

* vyvarujte se zásahů do struktury index.html, měňte jen class tak,
  aby stránka změnila svůj vhled s podoby devadesátých let do současné běžné podoby.

Pocet bodu (hvezdicek), desitek procent, ciselneho hodnoceni, slovniho hodnoceni
* 1, 20%, 5, nedostatecne
* 3, 30%, 4, dostatecne
* 5, 50%, 3, dobre
* 7, 70%, 2, velmi dobre (chvalitebne)
* 9, 90%, 1, vyborne
